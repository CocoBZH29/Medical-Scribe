# Medical-Scribe
Ce projet se place dans le contexte des cabines premium de TESSAN. Le but est d'implémenter un scribe médical qui pourra faire du speech-to-text en ajoutant de la valeur à l'anamnèse patient réalisée en amont de la téléconsultation. Le médecin sera ensuite en capacité de modifier ce compte rendu et de le valider.
